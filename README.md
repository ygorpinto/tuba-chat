## Projeto de um Real Time Chat utilizando Node.JS para treinar WebSockets, Socket.IO. 😳

Abrir localmente o server : 👈

git clone https://github.com/ygorpinto/tuba-chat.git

$ npm run start

abrir a localhost:3000 😌
